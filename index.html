StreamProtocol

Easy one-click streaming of media files in external players.

Download for Windows

What?

StreamProtocol makes it easier to get media to your media player. The only thing you need to do is change the links slightly or use an already compatible service.

The difference in links can be seen below:

http://www.example.com/video.mp4 downloads the file.
stream+
http://www.example.com/video.mp4 plays in VLC.

How to install

  • Download
  • Install
  • Select a media player in your new tray icon by right-clicking on it
  • Try out StreamProtocol by clicking here.

Supported Stream Players

VLC Media Player

VLC can be downloaded here and is detected out of the box.

Media Player Classic - HC

MPC-HC can be downloaded here and is detected out of the box.

Any Video Player

Most Video Players can play HTTP Links but are currently not auto-detected by StreamProtocol. You can just point StreamProtocol to the main executable for your favourite media player and enjoy video in it.

Supported Torrent Clients

Deluge Streaming

Currently, only Deluge through my plugin is supported directly. Information about my Deluge Streaming plugin can be found here. For this to work, you will need to have GTK UI with the plugin installed, running locally.
This plugin works best when used in combination with GreaseMonkey scripts to modify websites with Torrents.

Other Streaming Torrent Clients

There are plenty of streaming torrent clients out there. One example is Butter Project. These are currently not autodetected but you can just point StreamProtocol to their executable and everything should work.

Instructions

Installation

  • Download
  • Install
  • Right-click tray icon and select a player for HTTP and, optionally, one for Torrent (if you want to stream torrents)

View details »

Usage

  • Install GreaseMonkey (Firefox) or TamperMonkey (Chrome)
  • Install some addons for GreaseMonkey / TamperMonkey to modify your favourite websites
